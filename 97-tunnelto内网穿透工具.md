# **tunnelto** 内网穿透工具

这是一款非常受开发者欢迎的**开源内网穿透工具**（Reverse Proxy / Local Tunneling）。它的作用简单来说就是：**让你家里或公司内网的电脑，能被全世界访问。**

------

### 1. 它能解决什么问题？

假设你在本地电脑上开发了一个网站（地址是 `localhost:8000`），你想发给异地的朋友看看，或者测试微信支付的回调，但你没有公网 IP。 **Tunnelto** 就像在公网和你的内网之间挖了一条“隧道”：

- 它会分配给你一个类似 `https://xyz.tunnelto.dev` 的公网域名。
- 任何人访问这个域名，流量都会被转发到你的本地机器上。

### 2. 它的核心特点

- **极简：** 它是用 **Rust** 编写的，运行速度极快且占用资源极低（只有一个二进制文件）。
- **功能：** 支持 HTTP、HTTPS 转发，甚至可以转发 TCP 流量。
- **竞品：** 它是 **ngrok** 或 **frp** 的轻量级替代品。相比 ngrok，它的某些高级功能（如自定义子域名）在自建服务器上是免费的。

### 4. 它是怎么到你电脑上的？

它通常不会随 Windows 预装。如果你在电脑里发现了它，可能是因为：

1. **你是开发者：** 你之前为了调试代码，通过 `cargo` 或直接下载 `exe` 安装过。
2. **你用过某些集成包：** 某些本地开发环境（如 PHP/Node.js 的某些套件）可能集成了它作为内网穿透插件。

### 5. 可以删除吗？

- **完全可以。** 它不是系统组件，删掉它只会导致你无法通过该工具进行内网穿透。
- **删除方法：** 它通常是“绿色版”，没有安装程序。你只需要在任务管理器里结束进程，然后直接删除对应的 `tunnelto.exe` 文件即可。

------

**如果你不是开发者，却发现这个进程在后台高频连接网络，那就要留心了——因为它具备“反向访问”的能力，有时会被坏人用来绕过防火墙远程控制电脑。**
